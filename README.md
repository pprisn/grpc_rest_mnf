grpc_rest_mnf
=============

Микросервис для хранения и обработки запасных частей
----------------------------------------------------

### Технологический стек реализации микросервиса 
 go 1.13+
 grpc-rest-gateway
 protobuf 3
 postgres 9.6+

### Структура таблицы part
id, mnf_id, vendor_code, created_at, deleted_at

### Структура таблицы part_manufacturer
id, name, created_at
